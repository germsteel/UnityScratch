# Rotate 模块

本模块包含旋转相关的命令实现，用于控制对象的旋转。

## 文件说明

- **BaseRotateCommand.cs**: 旋转命令的基类，定义了旋转命令的共同行为

## 子模块

- **by**: 相对旋转命令，按相对角度旋转
- **to**: 绝对旋转命令，旋转到指定角度
